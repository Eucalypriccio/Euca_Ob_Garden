### 模型加载与 4-bit 量化配置

```python
from transformers import AutoModelForCausalLM, AutoTokenizer, BitsAndBytesConfig

model_id = "NousResearch/Llama-2-7b-chat-hf"

# 配置 BitsAndBytesConfig，指定 4-bit量化
bnb_config = BitsAndBytesConfig(
    load_in_4bit=True,                      # 启用 4-bit 量化
    bnb_4bit_use_double_quant=True,         # 启用嵌套量化
    bnb_4bit_quant_type="nf4",              # 量化类型（NormalFloat 4）
    bnb_4bit_compute_dtype=torch.bfloat16   # 计算数据类型，即使权重存储为 4-bit，实际计算会被解量化到这个数据类型
)

print("正在加载 tokenizer...")
tokenizer = AutoTokenizer.from_pretrained(model_id)
tokenizer.pad_token = tokenizer.eos_token
tokenizer.padding_side = "right"

print("正在加载并量化模型...")
model = AutoModelForCausalLM.from_pretrained(
    model_id,
    quantization_config=bnb_config,
    device_map="auto",
    torch_dtype=torch.bfloat16,
    trust_remote_code=True
)

print("tokenizer 和模型加载成功")
```

- `tokenizer`：分词器，将文本转换为一个 token 列表
- `pad_token`：用于填充序列中较短的序列，使其长度一致，便于批处理
- `eos_token`：end of sentence token，表示一个序列的结束
- `padding_side`：
	![[Pasted image 20250716204856.png]]
- `bfloat16`：Brain Floating Point 专为深度学习设计的一种 16 位浮点数格式
	- 1 位符号位
	- 8 位指数位
	- 7 位尾数位
- `trust_remote_code`：当加载模型配置文件时，从 Hugging Face 或其他在线资源下载

### 量化模型推理

```python
from transformers import pipeline
import time

# 创建一个文本生成 pipeline
generator = pipeline(
    "text-generation",
    model=model,
    tokenizer=tokenizer,
    torch_dtype=torch.bfloat16,
    device_map="auto"
)

prompt = "Tell me about the history of artificial intelligence in a few paragraphs."
print(f"\n用户输入：{prompt}")

print("正在使用量化模型生成文本...")
start_time = time.time()
outputs = generator(
    prompt,
    max_new_tokens=256,         # 最大生成 tokens 数量
    num_return_sequences=1,     # 返回序列的数量
    do_sample=True,             # 启用采样
    top_k=50,                   # 从 top_k 个最高概率的 token 中采样
    top_p=0.95,                 # 使用核采样（nucleus sampling），只从累积概率达到 top_p 的 token 中采样
    temperature=0.7,            # 控制生成文本的随机性，越低越保守，越高越有创意
    repetition_penalty=1.1,     # 惩罚重复的 token
)
end_time = time.time()

generated_text = outputs[0]["generated_text"]
print("\n--- 量化模型生成结果 ---")
print(generated_text)
print(f"\n量化模型推理时间: {end_time - start_time:.2f} 秒")
```

- 在输出分布上进行采样的策略
	- 温度采样 Temperature Sampling：在 softmax 中引入温度 $T$ 
	$$
	P_T(i) = \frac{\exp{(z_i/T)}}{\sum_{j=1}^V \exp{(z_j/T)}}
	$$
	- top-k sampling：保留概率最大的 k 个 token，从中采样
	- top-p sampling/nucleus sampling
		- 将输出分布从大到小排列
		- 计算累积概率，直到 $\sum_{i=1}^m P(i) \geq p$ 
		- 保留前 $m$ 个 token，归一化后从中采样
